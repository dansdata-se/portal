# Card

_Note: There exists a <LinkTo kind="components-card">helper component for cards</LinkTo>_

There are three types of cards: elevated, filled, and outlined.
These are based on the [guidelines from Material Design 3](https://m3.material.io/components/cards/overview).

Grids of cards should use the `gap-card-grid` utility class to ensure correct spacing.

## Clickable/Focusable Card

To make a card clickable/focusable, add the `tabindex` attribute.

Note that javascript is required to disable the focus effect when focusing child
elements, such as buttons. This is included in
the <LinkTo kind="components-card">`<Card>` helper component</LinkTo>
