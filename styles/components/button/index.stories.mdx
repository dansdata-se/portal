# Button

## Common Buttons

_Note: There exists a <LinkTo kind="components-button">helper component for buttons</LinkTo>_

_Icon Buttons should NOT use this style. See separate section for Icon Buttons._

Common buttons are created using the `.btn` class and `.btn-${variant}` modifiers.
The expected target element is `button`.

Direct children of the `button` element must be contained in an element:

```html
<!-- ✅ Do this ✅ -->
<button class="btn btn-text">
  <span>Click me!</span>
</button>

<!-- ❌ Not this ❌ -->
<button class="btn btn-text">Click me!</button>
```

The button supports `<img />` and `<svg />` icons:

```html
<button class="btn btn-text">
  <img src="someicon.png" />
  <span>Click me!</span>
</button>
```
