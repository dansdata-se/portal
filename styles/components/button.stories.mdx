# Button

## Common Buttons

There are five types of common buttons: elevated, filled, tonal, outlined, and text.
These are based on the [guidelines from Material Design 3](https://m3.material.io/components/buttons/overview).

Note: the design guidelines calls for extra size to be added to text buttons with icons.
To accommodate this, the last direct child of the `<button>` must be an element.
It is therefore suggested that you wrap the text inside the button with a `<span>`:

```html
<button class="btn btn-text">
  <img />
  <span>Click me!</span>
</button>
```
