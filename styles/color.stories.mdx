# Colors

Listing of the dansdata colors and their tailwindcss "tokens".

The dansdata design system is based on [material design 3](https://m3.material.io/)
and their system of
[design tokens](https://m3.material.io/foundations/design-tokens/overview).

All color tokens have a long form, e.g.: `md-sys-color-$(token)-light`.
Most color tokens also have a short form, e.g.: `$(token)`. The short form
without a `light`/`dark` suffix is preferred in-code. These are also designed
to adapt to the current dark mode setting:

```css
.bg-md-sys-color-$(token),
.bg-$(token) {
  @apply bg-md-sys-color-$(token)-light dark:bg-md-sys-color-$(token)-dark;
}

.text-md-sys-color-$(token),
.text-$(token) {
  @apply text-md-sys-color-$(token)-light dark:text-md-sys-color-$(token)-dark;
}
```

See also [./globals.css](./globals.css)
