# Colors

Listing of the dansdata colors.

The dansdata design system is based on [material design 3](https://m3.material.io/)
and their system of
[design tokens](https://m3.material.io/foundations/design-tokens/overview).

All colors have an associated css variable in [./material.css](./material.css).
Generally, the tokens provided by tailwindcss should be used over the raw css
variables:

```css
/* Prefer */
.my-custom-style {
  @apply text-primary;
}
/* Over */
.my-custom-style {
  @apply text-[rgb(var(--md-sys-color-primary))];
}
```
