# Typography

Listing of the dansdata text styles and their tailwindcss "tokens".

The dansdata design system is based on [material design 3](https://m3.material.io/)
and their system of
[design tokens](https://m3.material.io/foundations/design-tokens/overview).

All text tokens have a long form, e.g.: `md-sys-typescale-$(variant)-$(size)-$(property)`.
Most text tokens also have a short form, e.g.: `$(variant)-$(size)`. The short form
is preferred in-code.

## Prose

Inspired by [`@tailwindcss/typhography`](https://tailwindcss.com/docs/typography-plugin),
the `prose` class can be used to automatically make pieces of text beautiful without the need
for additional styling classes.

Using a surrounding tag with the `prose` class is preferred to setting margins manually per-page.
The `prose` class can also be added directly to an element to use its `prose`-styling:

```html
<span class="prose">
  <p>Lorem Ipsum</p>
</span>

<!-- Or -->

<p class="prose">Lorem Ipsum</p>
```

See [`prose.css`](./prose.css) for how these styles are set up.
